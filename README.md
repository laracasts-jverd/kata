# Kata using PHP Unit

Initialization
```sh
composer require phpunit/phpunit
composer dump-autoload # after adding psr-4 in composer.json
```

Execute tests with PHPUnit
```sh
./vendor/bin/phpunit tests
./vendor/bin/phpunit tests/RomanNumeralsTest.php
```
